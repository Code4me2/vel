# vel

Personal website for Vel Moon, published as a static GitHub Pages site.

## Local preview

No build step is required. Serve the repo root with any static file server, for example:

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.

## Deploy

The site is deployed from the `main` branch via GitHub Pages. Commit changes and push:

```bash
git push origin main
```
